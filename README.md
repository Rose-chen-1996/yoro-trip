# yoro-trip
Yoro Valley Camping Trip with Yata
# 養老渓谷 旅行しおり (2026.07.18 - 19)

スマートフォン（iPhone）での閲覧を最優先した、1カラムスクロール型のデジタル旅行しおりです。

## 🌲 デザインコンセプト
- **Snow Peak** の自然に溶け込む上質さ・アースカラー
- **無印良品** の無駄のないミニマリズム・余白美
- **ことりっぷ** の旅のワクワク感をそそる洗練されたレイアウト

## 📱 主な機能
- **レスポンシブ対応**: iPhone Safariに最適化。PCブラウザでも美しく中央に配置されます。
- **インタラクティブな持ち物リスト**: チェックを入れると自動で `LocalStorage` に保存され、ブラウザをリロードしても状態が維持されます。
- **ワンタップ Google Maps 連携**: 案内カードから直接ルートや場所を確認可能。

## 🚀 GitHub Pages で世界に公開する方法

このリポジトリをそのまま GitHub にアップロードするだけで、自分だけの旅行しおりWebサイトが完成します。

### ステップ 1: GitHubでリポジトリを作成
1. GitHubにログインし、右上の「**+**」ボタン ➔ **New repository** を選択。
2. Repository name に `yoro-trip` と入力。
3. Public（公開）にチェックを入れて、「**Create repository**」をクリック。

### ステップ 2: ファイルをアップロード
1. 画面中央付近にある「**uploading an existing file**」のリンクをクリック。
2. 本プロジェクトのファイル一式（`index.html`, `style.css`, `script.js`, `README.md`, および `images` フォルダ）をドラッグ＆ドロップ。
3. 画面下部の「**Commit changes**」ボタンをクリックして確定させます。

### ステップ 3: GitHub Pagesの設定
1. リポジトリ上部のメニューから「⚙️ **Settings**」を開きます。
2. 左サイドバーにある「**Pages**」をクリック。
3. **Build and deployment** 内の「Branch」項目で、`None` から `main` (または `master`) に変更し、右側のフォルダ指定を `/(root)` のまま「**Save**」をクリック。

### 🎉 完了！
1〜2分待ってからページをリロードすると、画面上部に以下のようなURLが発行されます：
`https://<あなたのユーザー名>.github.io/yoro-trip/`

このURLをスマホのLINEなどで共有したり、Safariの「ホーム画面に追加」をしておくことで、旅先でいつでもサクッと確認できるようになります。
